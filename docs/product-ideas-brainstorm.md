# 머니 파이프라인 아이디어 브레인스토밍

날짜: 2025-11-01

## 목표
- 사람들이 필요로 하는 서비스 100개 만들기
- 빠른 실행 (찍어내기)
- 코드 완전 재사용 가능한 구조
- 광고 수익 or Freemium 모델

## 핵심 원칙
1. **세상에 없거나 OR 10배 이상 나은 서비스**
2. **실제 니즈 기반**
3. **빠른 검증과 실행**

## 아이디어 후보들

### 1. PDF 도구 올인원
**니즈**: 병합, 분할, 압축
**문제점**:
- 기존 경쟁자 많음 (ilovepdf, smallpdf)
- 100배 낫기 어려움
- SEO 경쟁 심함

**차별화 시도**:
- 광고 없음
- 브라우저에서만 처리 (서버 업로드 없음)
- 특정 타겟 (한국 프리랜서, 부동산 등)

**평가**: 패스 (뾰족함 부족)

---

### 2. API 문서 자동 테스터
**니즈**: API 문서가 실제로 작동하는지 확인 귀찮음
**솔루션**:
- URL만 넣으면 OpenAPI/Swagger 자동 파싱
- 전체 엔드포인트 자동 테스트
- 깨진 API 즉시 알림

**10배 나은 포인트**: Postman보다 10배 빠름 (설정 불필요)
**수익**: B2B SaaS, 팀 플랜
**평가**: 유망 (개발팀 타겟)

---

### 3. README 자동 생성기
**니즈**: GitHub README 쓰기 귀찮음
**솔루션**:
- 코드 분석해서 자동 생성 (AI 활용)
- 설치법, 사용법 자동 추출
- Badge 자동 생성

**10배 나은 포인트**: 경쟁자 거의 없음
**수익**: Freemium
**평가**: 괜찮음

---

### 4. ⭐ ENV 파일 암호화 공유 (envshare)
**니즈**: .env 파일 팀원들과 안전하게 공유하기 어려움

**솔루션**:
```bash
$ envshare encrypt .env
→ https://envshare.dev/abc123

# 팀원이
$ envshare get abc123
→ .env 파일 생성 (1회용 링크)
```

**10배 나은 포인트**:
- 1회 조회 후 자동 삭제
- E2E 암호화
- 경쟁자 거의 없음
- 개발자들 100% 니즈

**수익 모델**:
```
무료:
- 1회용 링크
- 24시간 유효
- 파일 크기 1MB

Pro ($5/월):
- 7일 유효 링크
- 10MB
- 팀 기능
- Slack 연동

Team Plan ($29/월):
- 조직 대시보드
- 감사 로그
- SSO 연동
```

**광고 수익**: 어려움 (트래픽 적고 반복 사용 적음)
**대안**: Freemium or 오픈소스 + Sponsorship

**평가**: 가장 유망 (장기 파이프라인)

---

### 5. Lighthouse 자동 체크
**니즈**: 웹 성능 매번 수동 체크
**솔루션**:
- GitHub Action 한 줄로 자동화
- PR마다 성능 점수 댓글
- 무료 (Lighthouse 오픈소스)

**10배 나은 포인트**: 경쟁자 대부분 유료
**수익**: 오픈소스 or 프리미엄 기능
**평가**: 괜찮음

---

### 6. ⭐ 정규식 테스터 + AI 설명
**니즈**: 정규식 이해하기 어려움

**솔루션**:
```
정규식 입력 → AI가 한글로 설명
예: "^[a-z0-9]+@[a-z]+\.[a-z]{2,3}$"
→ "이메일 형식을 검증하는 패턴입니다"
```

**10배 나은 포인트**:
- 기존 테스터에 AI 설명 추가
- 초보 개발자 친화적

**수익**: 광고 수익 가능
- 높은 SEO 트래픽
- 반복 사용
- 체류시간 김

**평가**: 광고 수익 유망 (빠른 검증용)

---

### 7. 크론 표현식 생성기
**니즈**: cron 문법 어려움
**솔루션**: GUI로 생성 → "매주 월요일 오전 9시" → "0 9 * * 1"

**수익**: 광고
**평가**: 괜찮음 (SEO 트래픽)

---

### 8. 에러 로그 → 해결책 자동 검색
**니즈**: 에러 나면 구글링 반복
**솔루션**:
- 에러 메시지 복붙하면 Stack Overflow + GitHub Issues 자동 검색
- AI가 해결책 요약
- 브라우저 확장 or CLI

**평가**: 바이럴 가능성

---

### 9. 코드 스니펫 스크린샷 (Carbon 스타일)
**니즈**: 코드 예쁘게 공유
**솔루션**:
- 코드 → 이미지 생성
- SNS 공유용

**수익**: 광고 or 워터마크 제거 유료
**평가**: 트래픽 높음
